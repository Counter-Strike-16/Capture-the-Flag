SETTING UP THE CHALLENGES DEV VM


1. Install VirtualBox (https://www.virtualbox.org/wiki/Downloads)
2. Install Vagrant (https://www.vagrantup.com/downloads.html)
3. Enable Virtualization in BIOS
	For HP EliteDesk machines, hit F10 on startup and navigate to the 'Security'
	tab - enable the virtualization (VTx) setting
4. Navigate to the challenge_vm directory in cmd
5. Run the command 'vagrant up'
6. Challenges can now be accessed through links on the CTF platform!
